# Promedio Semanal del Clima en Python


## Descripción
Este proyecto compara dos enfoques de programación en Python: Programación Tradicional y Programación Orientada a Objetos (POO), mediante el cálculo del promedio semanal del clima.


## Objetivo
Aplicar y comprender las diferencias entre ambos paradigmas de programación a través de un ejemplo práctico.


## Archivos
- tradicional.py: Implementación usando programación estructurada.
- poo.py: Implementación usando programación orientada a objetos.


## Conclusión
La POO permite una mejor organización y reutilización del código, siendo más adecuada para proyectos grandes, mientras que la programación tradicional es más simple para problemas pequeños.
